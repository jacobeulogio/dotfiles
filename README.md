
# Dotfiles

Collection of my dotfiles for easy migration

## Installation

### Clone the Repo

```bash

git clone --recurse-submodules --remote-submodules https://github.com/jacobeulogio/dotfiles.git ~/dotfiles

```

### Install GNU Stow

```bash

sudo pacman -S stow

```

### Stow dotfiles

```bash

stow .

```
